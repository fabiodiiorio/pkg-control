Function Reference: strseq

Function File: strvec = strseq (str, idx)

Return a cell vector of indexed strings by appending the indices idx to the string str.

 
 strseq ("x", 1:3) = {"x1"; "x2"; "x3"}
 strseq ("u", [1, 2, 5]) = {"u1"; "u2"; "u5"}
